# Истоки

Интерактивное семейное древо, которое начинается с самых дальних известных предков. Первый запуск открывает чистое древо без демонстрационных людей. Записи можно добавлять, связывать двумя родителями и несколькими супругами, дополнять девичьей фамилией, местами рождения и смерти, фотографией с ручной подгонкой, редактировать и удалять. Иерархическая раскладка держит поколения на отдельных уровнях, братьев и сестёр — в общей ветви, а текущих супругов — компактной парой; бывшие супруги остаются самостоятельными карточками. Каждая карточка перемещается независимо — мышью, пальцем или кнопками точного сдвига в панели человека; линии родства следуют за новыми координатами. Изменения сохраняются в `localStorage` текущего устройства.

Перед публикацией настоящей истории семьи заранее решите, какие данные допустимо показывать публично.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Публикация в GitHub Pages настроена через `.github/workflows/deploy.yml` и запускается после push в `main`.
