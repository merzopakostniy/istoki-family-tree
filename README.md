# Истоки

Интерактивное семейное древо, которое начинается с самых дальних известных предков. Стартовая версия работает полностью в браузере: записи можно добавлять и редактировать, а изменения сохраняются в `localStorage` текущего устройства.

Все люди в исходных данных — демонстрационные. Перед публикацией настоящей истории семьи замените их своими сведениями и заранее решите, какие данные допустимо показывать публично.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Публикация в GitHub Pages настроена через `.github/workflows/deploy.yml` и запускается после push в `main`.
